# João Pedro Biguelini - Portfólio

Portfólio pessoal moderno e profissional que apresenta minha trajetória como desenvolvedor de software.

## 🚀 Tecnologias

- **Framework**: Next.js 15 (LTS)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📁 Estrutura de Diretórios

```
app/
├── components/          # Componentes reutilizáveis
│   ├── Header.tsx       # Navegação
│   ├── Hero.tsx         # Apresentação pessoal
│   ├── About.tsx        # Sobre mim
│   ├── Services.tsx     # Áreas de atuação
│   ├── Technologies.tsx # Stack técnico
│   ├── Differentials.tsx # Filosofia de trabalho
│   ├── Projects.tsx     # Portfólio
│   ├── CTA.tsx          # Chamada para contato
│   └── Footer.tsx       # Rodapé
├── globals.css          # Estilos globais
├── layout.tsx           # Layout principal
└── page.tsx             # Página inicial
```

## 🎨 Design

- **Cores principais**: Roxo (#7C3AED, #8B5CF6, #A855F7)
- **Fundo**: Escuro elegante (#0F0F23)
- **Estilo**: Premium, minimalista, focado em credibilidade
- **Animações**: Framer Motion com transições suaves

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar

```bash
# Desenvolvimento
npm run dev

# Build para produção
npm run build

# Iniciar em produção
npm start
```

### Componentes Globais

#### Glass Effect
Efeito de vidro com blur para elementos

```jsx
<div className="glass-effect">Content</div>
```

#### Gradient Text
Texto com gradiente de roxo

```jsx
<span className="gradient-text">Texto</span>
```

#### Glow Effects
Efeitos de brilho para destaque

```jsx
<div className="glow-purple">Content</div>
<div className="hover-glow">Hover me</div>
```
